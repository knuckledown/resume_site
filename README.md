* Flask
	* https://www.codewithharry.com/blogpost/flask-cheatsheet/

```
├── flaskr/ # SERVER CODE IS TOP LEVEL
│   ├── __init__.py
│   ├── db.py
│   ├── schema.sql
│   ├── auth.py
│   ├── blog.py
│   ├── templates/ # HTML PAGES
│   │   ├── base.html
│   │   ├── auth/
│   │   │   ├── login.html
│   │   │   └── register.html
│   │   └── blog/
│   │       ├── create.html
│   │       ├── index.html
│   │       └── update.html
│   └── static/ # CSS AND BROWSER JAVASCRIPT
│       └── style.css
```
